# Uribou & Buchi Cafe

2匹の子ぶたとゆっくり過ごす、架空のカフェをテーマにした学習・ポートフォリオ用Webサイトです。

## このフォルダについて

このフォルダが、2026年7月18日時点の「提出・公開に使用する完成版」です。

- 全6ページ構成（ページは今後増やさない方針）
- パソコン・タブレット・スマートフォン対応
- 画像をWebPへ変換し、不要素材を除外した軽量版が土台
- スクロール演出、フッター写真コラージュ、FAQ、予約フォームを実装
- 架空サイトであることをフッターとフォームに明記

## ページ構成

| ファイル | ページ内容 |
| --- | --- |
| `index.html` | TOP・営業日・紹介・動画・アクセス |
| `about.html` | カフェのコンセプト・予約案内 |
| `piggies.html` | ウリ坊とぶちの紹介 |
| `gallery.html` | 写真と動画 |
| `access.html` | 営業時間・所在地・地図 |
| `reserve.html` | 予約・お問い合わせフォーム・FAQ |

## 確認方法

1. フォルダ内の `index.html` をダブルクリックします。
2. 上部メニューと右下のReserveボタンから全ページを確認します。
3. 修正後はブラウザを再読み込みします。

## GitHub Pagesへ手作業でアップロードするとき

このフォルダ自体ではなく、**フォルダの中身をすべて**アップロードします。`index.html` が公開場所の一番上に見える状態が正解です。

アップロード後は、公開ページを再読み込みし、TOP・メニュー・Reserve・FAQ・フォームを確認してください。

## 画像・動画を差し替えるとき

同じファイル名・同じ保存場所で差し替えると、HTMLを書き換えずに更新できます。大きな画像はWebP、動画はMP4を基本にし、差し替え後にスマートフォンでも表示速度を確認してください。

主な動画ファイル：

- `hero-pig.mp4`
- `piggy-movie.mp4`

## 提出用の注意

- 住所、店舗、Instagram、予約はすべて架空です。
- 予約フォームは入力練習用で、外部への送信は行いません。
- Instagramは、架空アカウントではなくInstagramトップへ移動するデモリンクです。
- 公開前の確認項目は `docs/delivery-checklist-20260718.md` にまとめています。
