# v31 修正メモ（2026-07-20）

## 今回の対象

- 装飾イラストと文字の重なり
- 背景の上に表示する文字の読みやすさ

## 変更内容

1. 本文内の文字を装飾より前へ表示した
   - 見出し
   - 本文
   - 店舗情報
   - 表の文字
   - FAQ
   - 本文内のリンク
2. 文字へ半透明の細い白縁を追加した
3. 白縁の周囲へ、ごく薄い白い影を追加した
4. 動きによって別の重なりになる文章のまとまりも前面へ調整した
5. 装飾イラストの位置、大きさ、透明度、表示数は変更していない

## 白縁の設定

```css
-webkit-text-stroke: .25px rgba(255, 255, 255, .58);
```

白色を完全に表示せず、半透明にしている。文字が太く見えすぎず、背景イラストと重なった場合だけ輪郭が分かりやすくなる程度を目標とした。

## 手動で確認する項目

- TOPの文章と営業時間が装飾より前に見える
- About、Piggies、Gallery、Access、Reserveの文字が装飾に隠れない
- 白縁が太すぎたり、文字がぼやけたりしていない
- 動画上の白い文字が不自然に太くなっていない
- ボタンやリンクの文字が読みづらくなっていない
- パソコン、タブレット、スマートフォンで同じ印象になっている
