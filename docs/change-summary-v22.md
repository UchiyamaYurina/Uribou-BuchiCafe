# v22 仕上げ内容

軽量版v21の見た目と構成を維持し、提出物としての使いやすさと説明力を高めました。

## 全ページ共通

- 現在ページをメニューの色と下線で表示
- キーボード利用者向けの「本文へ移動」を追加
- キーボードでスマートフォンメニューを操作可能にした
- Escキーで開いたメニューを閉じられるようにした
- 選択中の場所が見えるフォーカス表示を追加
- 左下に「動きを止める／動きを再生」ボタンを追加
- 端末の「視差効果を減らす」設定を尊重
- ファビコン、テーマカラー、正規URL、SNS共有用の文章情報を追加

## Reserve

- 予約期限と人数の案内をフォーム直前へ追加
- 予約を選択した場合のみ、希望日・希望時間・人数を必須化
- 予約期限との矛盾を避けるため、希望日は翌日以降を選択可能にした
- 入力不足がある場合、最初の未入力項目へ移動
- 入力を修正した際は古い確認メッセージを非表示にするよう改善

## TOP・Gallery

- 動画に読み上げ用の説明または装飾動画の指定を追加
- 架空のInstagramリンクであることを画面上でも明示
- 存在しない架空アカウントではなく、Instagramトップへ移動する安全なデモリンクへ変更
