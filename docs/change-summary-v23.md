# v23 予約案内の配置調整

## 変更内容

- イラスト付きの予約案内をReserveページからAboutページへ移動
- Aboutの「予約する」ボタンからReserveの入力欄へ直接移動する構成へ変更
- Reserveページは、予約フォームとFAQを中心とした簡潔な構成へ整理
- AboutとReserveの検索用説明文も、新しいページ内容に合わせて更新

## 変更した理由

Aboutでカフェの考え方を知った後に予約案内を表示することで、「このカフェへ行ってみたい」という気持ちから予約へ自然につながります。Reserveは手続きをするページとして役割が明確になり、入力欄にも早く到達できます。
