# v28 修正メモ（2026-07-19）

## 今回の対象

- Reserveページの固定Reserveボタン
- ReserveページのFAQ装飾
- TOPページの背景イラストと文字の重なり
- TOPページを狭い画面で表示したときの文章幅

## 変更内容

1. Reserveページだけ、右下の固定Reserveボタンを外した
   - 予約ページ内では移動先が同じページになるため
   - TOP、About、Piggies、Gallery、Accessの固定Reserveボタンは残している
2. FAQに既存のグリーン（sage）を追加した
   - FAQ全体に薄いグリーンの背景
   - 各質問を白いカードに変更
   - 質問の「Q」と開閉記号にグリーンを使用
3. TOPの文字を背景イラストより前に表示するよう調整した
   - イラストの位置、大きさ、透明度、表示数は変更していない
4. 画面幅が狭いとき、TOPの文章を少し早めに折り返すよう調整した

## 手動で確認する項目

- Reserveページ右下に固定Reserveボタンがない
- 他の5ページには固定Reserveボタンがある
- FAQをクリックすると回答が開閉する
- FAQの文字がグリーンの装飾に隠れない
- TOPを横幅1050px前後から狭くし、文章が背景イラストより前に見える
- TOPのパソコン版とスマートフォン版に大きな崩れがない
