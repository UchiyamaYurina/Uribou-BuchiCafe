## 学んだこと

- フッターでは、情報を左・中央・右の役割に分けると整理しやすい
- `position: fixed`を使うと、スクロール中もボタンを同じ位置に表示できる
- 画像の余白は、CSSだけでなく画像ファイル自体にも含まれることがある
- 背景、角丸、影が重なると、表がカードのように見えやすい
- CSSは同じ指定が複数あると、記述順によって表示が変わる
- デザイン比較では前の版を残すと、安全に戻すことができる
- GitHubへはZIPファイルではなく、展開した中身をアップロードする
